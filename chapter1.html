1. Analytical Solution

An analytical solution is a solution to a mathematical model or equation that is derived using algebraic manipulations, calculus, and other standard mathematical techniques. It provides exact values and is typically faster and more accurate than numerical solutions when available.

Example of Analytical Solution:

For a simple linear equation: y = 2x + 3, we can solve for `y` analytically given any value of `x`.

2. Numerical Solution

A numerical solution approximates the solution to a mathematical problem using numerical methods and algorithms. While analytical solutions provide exact results, numerical solutions can be more practical for complex or unsolvable equations.

3. Error Analysis

Error analysis involves studying the errors that arise when solving problems numerically. Errors can result from approximations, truncations, or limitations in the methods used.

Types of Errors:

  • Truncation Error: Caused by approximating a mathematical procedure (e.g., replacing an infinite series with a finite number of terms).
  • Round-off Error: Arises when numbers are rounded to a limited precision.
  • Discretization Error: Introduced when continuous functions are approximated with discrete values.

4. Accuracy

Accuracy refers to how close a numerical solution is to the true value. A higher accuracy indicates that the solution is very close to the exact or real solution.

Example of Accuracy:

If we have a true value of y = 5.2 and our numerical solution is y = 5.3, the accuracy is determined by the difference between these values: 5.3 - 5.2 = 0.1.

5. Precision

Precision refers to the number of significant digits or decimal places used in a numerical calculation. A higher precision leads to more digits in the result, reducing the likelihood of round-off errors.

6. Roundoff Error

Roundoff error occurs because computers can only store a finite number of digits. This limitation can cause small discrepancies when performing arithmetic operations, especially with large or very small numbers.

Example of Roundoff Error:

For example, when calculating 1/3, the exact result is 0.333333..., but computers may store it as 0.3333333, introducing a small roundoff error.
